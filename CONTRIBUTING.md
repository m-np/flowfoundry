## Getting started
1. Fork the repo and create a feature branch:
   ```bash
   git checkout -b feat/my-change
   ```

2. Create a Conda env and install dev deps:
    ```bash
    conda env create -f environment.yml
    conda activate flowfoundry
    pip install -e ".[dev]"
    ```
